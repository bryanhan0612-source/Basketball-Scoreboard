# Basketball-Scoreboard
A simple web-based scoreboard for tracking the scores of two basketball teams: HOME and AWAY. Users can increment scores with buttons and reset the scoreboard when needed

Features:

- Increment Scores – Add 1, 2, or 3 points to each team with dedicated buttons.
- Reset Scoreboard – Reset both HOME and AWAY scores to 0.

Technologies Used:

- HTML
- CSS
- JavaScript

How to Use
1. Open index.html in a web browser or use https://bprojectb.netlify.app/
2. Click +1, +2, or +3 buttons under HOME or AWAY to increase the corresponding team’s score.
3. Click RESET to set both scores back to 0.
